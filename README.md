# MERN Expense Tracker

A simple full-stack MERN (MongoDB, Express, React, Node.js) application to track personal expenses.

## 🚀 Features

- Add and manage expenses
- View expense list
- Simple and clean UI
- Full-stack integration (React + Node.js + MongoDB)

## 🛠️ Tech Stack

### Frontend

- React
- Axios
- Tailwind CSS
- React Hooks (useState, useEffect)

### Backend

- Node.js
- Express.js

### Database

- MongoDB

---

## 📁 Project Structure

```
/client    -> React frontend
/server    -> Node/Express backend
```

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/arunk220895-codes/expense-tracker.git
cd mern-expense-tracker
```

### 2. Setup Backend

```
cd server
npm install
```

Create a `.env` file in the `/server` folder:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
```

Run backend:

```
npm start
```

---

### 3. Setup Frontend

Open a new terminal:

```
cd client
npm install
npm start
```

## 🌐 API Endpoints (Example)

- GET /expenses → Get all expenses
- POST /expenses → Add new expense
